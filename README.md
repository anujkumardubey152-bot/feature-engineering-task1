# feature-engineering-task1


## Objective
This project demonstrates data preprocessing and feature engineering on the Titanic dataset to prepare it for machine learning.

## Dataset
Titanic Dataset

## Tasks Completed
- Loaded the dataset using Pandas
- Performed Exploratory Data Analysis (EDA)
- Handled missing values
- Encoded categorical variables
- Scaled numerical features
- Created a new feature (FamilySize)
- Split data into training and testing sets
- Exported the processed dataset

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

- raw_dataset.csv
- processed_dataset.csv
